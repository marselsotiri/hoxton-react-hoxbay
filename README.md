
# Hoxbay

This is an E-commerce App you add, delete, patch products to the chart.


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
Start JSON server

```bash
  json-server --watch db.json --port
```
